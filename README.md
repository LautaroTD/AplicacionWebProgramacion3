# AplicacionWebProgramacion3

Creador: Lautaro Tamborini.
Este sistema web busca ser una pequeña e intuitiva enciclopedia donde se guarda Plantas, Suelos y Fertilizantes.
Cuenta con autentificacion simple basado en nombre de cuenta y constraseña y 5 roles distintos, basico, Admin, Admin de Planta, Admin de Suelo, y Admin de Fertilizante.
Registrarse genera un nuevo usuario Basico.
Basico solo puede ver los registros existentes de Plantas, Suelos y Fertilizantes.
Admin tiene acceso al CRUD de usuarios.
Admin de Planta tiene acceso al CRUD de Plantas.
Admin de Suelo tiene acceso al CRUD de Suelos.
Admin de Fertilizante tiene acceso al CRUD de Fertilizante.
